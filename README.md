# Predicting SpaceX Falcon 9 First Stage Landing Success

## Project Overview
This project predicts whether the Falcon 9 first-stage booster will land successfully using historical launch data from SpaceX. The goal is to support cost estimation, risk assessment, and mission planning.

## Contents
- `data/` - Dataset used in the project
- `notebooks/` - Jupyter notebooks for all steps:
  1. Data Collection
  2. Exploratory Data Analysis (EDA)
  3. Feature Engineering
  4. Model Training & Evaluation
  5. Predictive Analysis
- `scripts/` - Python scripts for each major step
- `requirements.txt` - Python packages required

## Key Features
- Exploratory data analysis with visualizations
- Interactive dashboards with Plotly Dash
- Predictive modeling using Logistic Regression, Decision Tree, Random Forest, and SVM
- Feature importance analysis
- Folium map of launch sites

## How to Run
1. Clone the repository:
git clone https://github.com/YourUsername/SpaceX_Falcon9_Capstone.git

2. Install required packages:

pip install -r requirements.txt

3. Open the Jupyter notebooks in `notebooks/` to reproduce the analysis.
